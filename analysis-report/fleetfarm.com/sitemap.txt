## URL Pattern Analysis

### Category Structure
- Format: `/category/{category-name}/_/N-{numeric-id}`
- Hierarchical: Categories can have multiple levels
- IDs appear to be system-generated unique identifiers

### Product URLs
- Format: `/detail/{product-slug}/{sku}?bc={category-id}`
- SKU Format: 13-digit numeric (0000000395713)
- bc Parameter: Links back to category ID

### Search URLs
- Format: `/search?Ntt={search-term}`
- Ntt Parameter: Product codes or search terms
- Additional filtering possible with N parameters

### Brand Pages
- Format: `/brand/{brand-name}/_/N-{brand-id}`
- Some include category filters: `/brand/{brand}/hunting/_/N-{brand-id}+{category-id}`

- No XML sitemap accessible (blocked by Cloudflare)

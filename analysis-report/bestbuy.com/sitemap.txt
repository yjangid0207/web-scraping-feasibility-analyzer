## Key Observations for Scraping:

3. Category structure follows hierarchical pattern with IDs
4. Search functionality uses JSP with query parameters
5. All endpoints protected by advanced anti-bot system
6. Geographic location affects product availability and pricing
7. HTTP access completely blocked - browser automation required
